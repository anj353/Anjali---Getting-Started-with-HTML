HTML comments are enclosed within <!-- and --> tags. Anything between these tags is treated as a comment and is ignored by web browsers when rendering the page. 

The syntax for creating an HTML comment is as follows:

<!-- This is an HTML comment -->

Example:

<body>
    <header>
        <h1>Welcome to My Website</h1>
        <!-- This is a comment explaining the purpose of the header element -->
        <p>Explore our content and services.</p>
    </header>

    <nav>
        <!-- Navigation links go here -->
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
            <!-- <li><a href="#">Services</a></li> -->
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>

    <main>
        <h2>About Us</h2>
    </main>

</body>
